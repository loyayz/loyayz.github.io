PaperMod 已集成 Fuse.js 实现搜索功能，本篇记录下如何使用

1

        includeMatches: true
        shouldSort: true
        location: 0
        distance: 1000
        threshold: 0.4
        minMatchCharLength: 0
        keys: ["title", "permalink", "summary", "content"]
